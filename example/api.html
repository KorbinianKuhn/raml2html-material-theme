Example API documentation version v1

https://{environment}.example.com/api/example/{version}

  • environment: (one of stg, dev, test, prod) (required)
  • version: v1 (required)
Getting Started

This is an example API to test all raml functionalities and the material theme. Check the /users POST route to get a complex modal with many information.

      Type: object

      Properties:
      • email: string (required)
        Email address of the user.
      • password: string (required) | minLength: 6 | maxLength: 20
        Password of the user.
      • age: integer | minimum: 10 | maximum: 110
        Age of the user.
      • gender: (one of active, inactive)
        Status of the user.
      • hobbies: array[string]
        Hobbies of the user.
      • name: object
        Name of the user.
        • firstname: string
          First name.

          Example

          Jane
        • lastname: string
          Last name.

          Example

          Doe

}
